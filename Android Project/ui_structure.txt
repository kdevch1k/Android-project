Budget Buddy - Структура UI

1. MainActivity (Главный экран)
   Файл: activity_main.xml
   Элементы:
   - AppBarLayout с Toolbar
   - TextView для отображения текущего баланса
   - CardView с кратким обзором (доходы/расходы за месяц)
   - RecyclerView для списка транзакций
   - Две FloatingActionButton:
     * Красная для быстрого добавления расхода
     * Зеленая для быстрого добавления дохода
   
   Особенности:
   - Баланс меняет цвет в зависимости от значения (зеленый/красный)
   - Транзакции группируются по дате
   - Pull-to-refresh для обновления данных
   - Быстрый доступ к часто используемым категориям

2. QuickInputActivity (Быстрый ввод)
   Файл: activity_quick_input.xml
   Элементы:
   - Top-5 последних категорий (горизонтальный список)
   - Большое поле ввода суммы
   - Цифровая клавиатура с калькулятором
   - Большая кнопка "Готово"
   - Свернутая панель для доп. опций (дата, заметка)
   
   Особенности:
   - Автоматический фокус на поле суммы
   - Вибрация при нажатии кнопок
   - Анимации при разворачивании доп. опций
   - Быстрое закрытие жестом вниз

3. AddTransactionActivity (Полный ввод транзакции)
   Файл: activity_add_transaction.xml
   Элементы:
   - Toolbar с кнопками "Отмена" и "Сохранить"
   - RadioGroup для выбора типа (доход/расход)
   - EditText для суммы с поддержкой калькулятора
   - Spinner для выбора категории
   - EditText для заметки
   - DatePicker для выбора даты
   
   Особенности:
   - Валидация суммы в реальном времени
   - Динамическая смена категорий при смене типа
   - Автоматическая установка текущей даты

4. Адаптер списка транзакций
   Файл: item_transaction.xml
   Элементы:
   - ImageView для иконки категории
   - TextView для названия категории
   - TextView для суммы
   - TextView для заметки (если есть)
   - TextView для даты
   
   Особенности:
   - Разные цвета для доходов и расходов
   - Форматирование суммы с разделителями
   - Сокращение длинных заметок
   - Свайп для быстрого удаления

5. Общие стили
   Файл: styles.xml
   - Основные цвета:
     * colorPrimary: #2196F3 (синий)
     * colorPrimaryDark: #1976D2
     * colorAccent: #4CAF50 (зеленый)
     * colorExpense: #F44336 (красный)
     * colorIncome: #4CAF50 (зеленый)
   
   - Текстовые стили:
     * Заголовок: 20sp, bold
     * Сумма: 18sp, medium
     * Обычный текст: 16sp
     * Дополнительный текст: 14sp, gray

6. Анимации
   - Переход между экранами
   - Появление клавиатуры
   - Обновление баланса
   - Добавление новой транзакции в список
